# Paw Hyprland

Dark Hyprland config with neon-magenta borders, rounded corners, and a custom
`pawCurve` bezier for all window animations. Pair it with `paw-waybar` and
`paw-rofi` for the full look.

## Install

```
cp hyprland.conf ~/.config/hypr/themes/paw-hyprland/hyprland.conf
# then in ~/.config/hypr/hyprland.conf:
source = ~/.config/hypr/themes/paw-hyprland/hyprland.conf
```

Restart Hyprland (`hyprctl reload` or log out/in) to apply.
